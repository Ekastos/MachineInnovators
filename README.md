## Monitoraggio della Reputazione Online di MachineInnovators Inc.

Questo Space ospita un'applicazione di analisi del sentiment per monitorare la reputazione online.

L'applicazione utilizza il modello `cardiffnlp/twitter-roberta-base-sentiment-latest` per classificare i testi in sentiment `positivo`, `neutro` o `negativo`.
